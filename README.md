# VectorTrader
量化研究。
回测模式下，用户需要配置策略所在路径，策略回测结果保存路径。
模拟模式下，用户需要配置策略所在路径，模拟结果每日更新路径，持久化路径。

# Installation
1. 向site-packages中添加VectorTrader路径文件pth.
2. 安装requirments中的需求
3. 在data_source中的mixed_data_source中根据Readme进行调整。
4. 向系统环境变量中添加mod地址
